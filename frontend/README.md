# PrepWise — Frontend

This folder will hold the PrepWise web client (planned).

The frontend will talk to the [backend](../backend) REST API over JWT-authenticated
HTTP. See the [backend README](../backend/README.md) for the API reference, roles,
and the local base URL (`http://localhost:8084`).

> Nothing has been scaffolded yet. Add your framework of choice (e.g. React + Vite)
> here and point it at the backend's `app.cors.allowed-origins` (defaults include
> `http://localhost:3000` and `http://localhost:5173`).
